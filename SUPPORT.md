# Support

## Product and account support

For KXCO products, accounts, and commercial questions, contact us through
[kxco.ai](https://kxco.ai) or email **hello@kxco.ai**. Please don't use
GitHub issues for account or commercial matters.

## Bugs and feature requests

Found a bug in one of these packages, or missing a capability? Open an issue
on the relevant repository using the provided template. We triage weekly.

## Security vulnerabilities

Never report security issues publicly. Use the repository's
**Security → Report a vulnerability** tab or email **security@kxco.ai**.
See [SECURITY.md](SECURITY.md) for scope and response times.

## Documentation

Each package's README covers installation and usage. Developer documentation
and the network explorer live at [chain.kxco.ai](https://chain.kxco.ai); the
[organization page](https://github.com/KnightsbridgeAIQ) lists the full stack.
